6. 錯誤處理與狀態碼

6.1 ESB 回應代碼

codesrvcode說明
019000成功 - 平台同步回覆
其他非 000失敗 - 查看 description

6.2 ERP 執行狀態

status/@code說明
0執行成功
非 0執行失敗，查看 description

6.3 MES 處理狀態

code說明MES 內部
200成功ISOK = true
500失敗ISOK = false

6.4 常見錯誤情境

錯誤訊息可能原因處理建議
工單不存在moType-moCode 錯誤檢查工單號碼格式
數量超過餘量累計入庫超過工單數量確認工單可入庫餘量
倉庫不存在moveInCode 無效檢查倉庫設定
Return result parsing error!ERP 回傳格式異常或連線失敗檢查 ESB/ERP 連線狀態
TimeoutERP 處理超時檢查 ERP 系統狀態，考慮補送
